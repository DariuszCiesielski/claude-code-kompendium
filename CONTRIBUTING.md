# Contributing

We welcome contributions to make this guide even better!

## How to Contribute

1. **Fork** this repository
2. Create a branch for your changes
3. Write content **in Polish** with proper diacritics (ą, ć, ę, ł, ń, ó, ś, ź, ż)
4. Submit a **Pull Request** with a clear description

## What We Need

- New tips and techniques for Claude Code
- Corrections and updates for existing chapters
- New example CLAUDE.md files for different project types
- Translations of code examples to Polish context

## Guidelines

- Keep examples practical and tested
- Include code blocks with proper syntax highlighting
- Reference official Anthropic documentation where applicable
- No API keys or secrets in examples
