# Wstęp {.unnumbered}

Kompendium Claude Code to najobszerniejszy polskojęzyczny przewodnik po oficjalnym CLI Anthropic do programowania z AI. Od pierwszych kroków po zaawansowane wzorce multi-agentowe.

## Dla kogo jest ta książka?

- Programiści zaczynający pracę z Claude Code
- Zespoły wdrażające AI w procesie developerskim
- Zaawansowani użytkownicy szukający optymalizacji
- Każdy, kto woli uczyć się po polsku

## Jak czytać?

Książka jest podzielona na cztery części:

1. **Podstawy** — instalacja, komendy, system skilli
2. **Zaawansowane techniki** — optymalizacja tokenów, projekty, automatyzacje, debugowanie
3. **Multi-agent** — wzorce delegacji, frameworki
4. **Integracje i wzorce** — MCP, najlepsze praktyki, wzorce promptów

Możesz czytać po kolei lub przeskakiwać do interesujących Cię rozdziałów.
